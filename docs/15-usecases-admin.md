# 15. 管理機能ユースケース

## UC-M001: 権限を設定する

1. 管理者がユーザー一覧を開く
2. 対象ユーザーを選択する
3. ロールを設定する
4. 権限変更イベントを記録する

## UC-M002: 申請種別を管理する

1. 管理者が申請種別一覧を開く
2. 申請フォーム項目を設定する
3. バックオフィスタスク生成有無を設定する
4. 保存する

## UC-M003: 監査ログを確認する

1. 管理者が監査ログ画面を開く
2. ユーザー、期間、対象種別で検索する
3. 操作履歴を確認する
4. 必要に応じてCSV出力する

## 実装上のポイント

- 権限管理 (UC-M001) は [UC-002 MS365ユーザーを同期する](./06-usecases-auth.md#uc-002-ms365ユーザーを同期する)
  で上書きされない、アプリ独自のロールとして管理する。
- 監査ログ (UC-M003) は `stored_events` を主な参照元とし、UIで検索しやすい形の
  Projection (例: `audit_log_entries`) を用意する。
