# 14. CSV出力ユースケース

## UC-E001: 勤怠CSVを出力する

1. 管理者が対象月を選択する
2. 対象社員を絞り込む
3. CSV出力する
4. 出力履歴を記録する

## UC-E002: 経費CSVを出力する

1. 経理担当者が対象期間を選択する
2. 承認済み・支払予定の精算を抽出する
3. CSV出力する
4. 出力履歴を記録する

## 実装上のポイント

- 出力操作は `export.created` イベントとして記録し、誰がいつ何を出力したかを追跡できるようにする。
- 給与計算ソフト・会計ソフトへの取込を想定し、CSVフォーマット(列順・エンコーディング・
  区切り文字)は取込先ソフトの仕様に合わせて設定可能にする。
- 締め後の勤怠CSV(UC-E001)は [UC-A011 月次勤怠を締める](./07-usecases-attendance.md#uc-a011-管理部が月次勤怠を締める)
  の後にのみ出力可能とする。
