# 1. システム概要

Laravel + MySQL + XSERVER 環境で、汎用的な勤怠管理・申請管理・バックオフィス処理管理システムを
構築する。

本システムは SES 専用ではなく、一般的な中小企業・受託会社・シフト勤務会社でも利用できる
社内業務基盤とする。

コミュニケーションや掲示板機能は Microsoft Teams に任せ、本アプリは以下に集中する。

- 勤怠記録
- 勤怠集計
- 有給管理
- 各種申請
- 承認
- バックオフィス処理
- 添付ファイル管理
- CSV出力
- 監査ログ
- Teams通知

Teams は通知先としてのみ使う。チャット、掲示板、お知らせ機能は作らない。
